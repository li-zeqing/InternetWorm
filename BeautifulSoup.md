# Beautiful Soup

> **Attributes**
>
> > 1. 一个tag可能有很多个属性，tag的属性操作方法和字典相同
> >
> > 2. `.attrs` 获取属性 返回为 class : value
> > 3. tag的属性可以被添加，删除和修改 
> >
> > 

> **NavigableString**
>
> > 1. 字符串常被包含在tag内，用`NavigableString`类来包装
> > 2. tag中包含的字符串不能编辑，但可以被替换 用`replace_with()`方法

> **Comment**
>
> > `Comment`是文档的注释部分 
> >
> > `Comment` 对象是一个特殊类型的`NavigableString`对象
